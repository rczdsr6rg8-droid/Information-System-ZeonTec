## Как скопировать код лендинга

Вы можете забрать исходники несколькими способами:

1. **Скачать ZIP c GitHub**
   - Откройте репозиторий на GitHub.
   - Нажмите **Code → Download ZIP** и распакуйте архив.

2. **Клонировать через Git**
   - Убедитесь, что установлен Git.
   - Выполните в терминале: `git clone <ссылка-репозитория>`
   - Перейдите в папку проекта: `cd Information-System-ZeonTec`

3. **Скопировать отдельные файлы вручную**
   - Откройте нужный файл (например, `index.html`, `styles.css` или `script.js`) на GitHub.
   - Нажмите кнопку **Raw**, затем выделите всё содержимое и вставьте в локальный файл с тем же именем.

После загрузки файлов вы можете открыть `index.html` двойным щелчком в браузере или запустить локальный сервер (например, `npx serve .`) и увидеть лендинг с обновлениями.
